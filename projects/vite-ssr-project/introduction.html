Side Project

  • react-gpt-cli

    React GPT CLI는 React 애플리케이션 개발의 효율성을 극대화하기 위한 CLI 도구입니다. 현재 베타 버전이며 시험적인 기능들이 포함되어 있습니다.
    GPT4 Turbo를 이용해 프로젝트를 분석 후 챗 기능, 자동 문서 생성, 컴포넌트 생성, commit 추천,프로젝트 초기화와 설정을 단순화하는 기능을 제공합니다. 이 도구는 개발자의 시간을 절약하고, 초기 개발 부담을 줄입니다.
