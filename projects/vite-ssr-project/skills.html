기술 및 역량 요약

  • 다양한 기술 스택: React, Vue, Node.js, Redis, MySQL, PHP, Laravel, Docker 등 다양한 기술을 사용한 경험이 있습니다. 이에 어떠한 프로젝트에도 유연하게 대응할 수 있습니다.
  • 리더십: 여러 회사에서 프론트엔드 파트 리더로서의 역할을 수행하였고, 팀과 프로젝트를 관리할 수 있습니다.
  • 프로젝트 다양성: C2C, SEO, 관리자 대시보드, 모바일 앱 등 다양한 분야에서의 프로젝트 경험이 있습니다.
  • 프로세스 최적화: 배포 플로우 체계화, Monorepo 셋팅, Terraform을 이용한 인프라 자동화 등을 통해 개발 프로세스를 최적화한 경험이 있습니다.
  • 디자인 시스템 및 상태 관리: Storybook으로 디자인 시스템을 구축하고, Recoil와 자체 개발한 'supplier'로 상태 관리를 효과적으로 수행한 경험이 있습니다.
  • 서비스 완전 주기 경험: 서비스 개발부터 유지보수, 그리고 서비스 종료까지의 전체 주기에 대한 경험이 있습니다.
  • 문제 해결 능력: 다양한 기술과 도구를 사용하여 문제를 해결하는 능력이 있습니다. (예: git-ftp, Cloudfront + Lambda Edge 등)
  • 지속적 학습과 성장: 새로운 기술과 도구를 계속적으로 습득하고, 신입 개발자를 위한 스터디를 진행하는 등 지속적으로 성장하고 있습니다.
  • 커뮤니케이션 및 문서화: 배포 현황을 노션으로 공유하고, CS 가이드를 제작하는 등의 커뮤니케이션 및 문서화 능력도 갖추고 있습니다.

HTML/CSS

  • 웹 접근성과 Semantic Markup

    HTML 작성 시 Semantic Markup을 철저히 고려합니다. 이는 코드 가독성과 웹 접근성을 높이는 데 중요한 역할을 합니다.

  • CSS 스타일링

    CSS 작성은 개발자 친화적이면서도 간결하게 유지합니다.

    각 HTML 요소를 명확하게 구분할 수 있도록 코드를 구조화합니다.

  • CSS Preprocessors과 CSS-in-JS

    Emotion과 Styled Component를 활용한 프로젝트 설계 및 구현 경험이 있습니다.

    CSS Module과 SCSS를 조합하여 효율적인 스타일링을 선호합니다.

  • 크로스 브라우저 지원

    Chrome, Safari, Firefox 등 다양한 브라우저를 지원하는 웹 애플리케이션을 개발할 수 있습니다.

  • 웹 최적화 및 SEO

    favicon, Open Graph Protocol을 포함하여 SEO를 적용한 프로젝트 경험이 있습니다.

TypeScript

  • React 프로젝트에서 TypeScript를 능숙하게 사용합니다.
  • 객체지향 프로그래밍에 익숙하며, 상속과 같은 클래스 타입을 적절하게 활용합니다.
  • Generic, Overloading, Tuple 등 고급 문법을 소화할 수 있습니다.

JavaScript

  • ES2015 이상의 JavaScript 문법을 능숙하게 사용합니다.
  • Vanilla JavaScript로 DOM 조작에 익숙하며, 고차 함수와 비동기 처리에 능숙합니다.

React

  • React 기반의 CSR/SSR 프론트엔드 개발에 익숙하고, 리렌더링 최적화, 번들 경량화 등 기본적인 웹 성능 최적화 경험이 있습니다.
  • React Hooks와 Context API를 활용하여 적절한 상태 관리를 구현합니다.
  • 스니펫, 탬플릿을 활용하여 관리 쉽고 유지보수 용이하게 Folder Structure를 구성합니다.
  • Redux를 활용한 상태 관리 경험 있으며, 레거시 코드 축적을 최소화합니다.
  • Container/Presentational 패턴을 이해하고, 코드 구조를 명확하게 분리합니다.

서버 개발

  • Express.js를 활용하여 간단하게 RESTful API를 작성할 수 있습니다.
  • 라우팅 규칙을 적절하게 사용하여 경로를 작성합니다.

    http 프로토콜에 기반하여 심플하고 이해하기 쉽고 확장성있는 엔드포인트를 선호합니다.

  • JWT 인증을 사용하여 브라우저 쿠키에 저장하고 세션 처리를 할 수 있습니다.

DevOps

  • 자동화 프로세스를 만들어서 팀 내 생산성을 높힙니다.

    코드리뷰 자동화 프로세스, 테라폼(Terraform)을 통해 인프라스트럭처를 선언적인 코드로 작성하고 관리

  • AWS EC2, ECS, S3 등의 인프라 요소를 사용해본 경험이 있습니다.
  • Jenkins, Docker를 사용하여 개발 환경을 구축할 수 있습니다.
  • Github Action, docker compose, AWS S3 사용하여 무중단배포를 구축할 수 있습니다.
  • Lambda 등 AWS의 서버리스 인프라에 익숙하고, 프론트엔드에서 필요한 수준의 서버 또는 인프라를 구성하고 유지보수할 수 있습니다.

    이를 위해 기능 로직, 컴포넌트 마다 최대한의 문서화를 제공하기 위해 노력합니다.
