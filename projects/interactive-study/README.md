# interactive-study

Interactive Developer 님의 유튜브를 보고 실습 진행한 예제 프로젝트

- bouncyball: 튕기는 볼
- mouseRect: 줄에 끌리는 박스
- shapeRotate: 회전하는 도형
- sheepHills: 언덕들을 오르는 양떼

## 추가 실습

base 폴더를 복사해서, 진행할 실습으로 이름 변경 후 package.json scripts에 `lite-server --baseDir='projectName'"` 추가
